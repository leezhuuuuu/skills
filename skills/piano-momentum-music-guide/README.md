# Piano Momentum Music Guide

> **Source**: This skill is derived from [anthropics/skills PR #178](https://github.com/anthropics/skills/pull/178) by @cherishwins.

钢琴学习和音乐理论综合指南。

## 内容概览

| 章节 | 内容 |
|------|------|
| 钢琴基础 | 键盘布局、手位、姿势、音符 |
| 练习技巧 | 慢练、分块练习、视奏、听音训练 |
| 乐理 | 音程、和弦、节奏、曲式 |
| 学习路径 | 初学→进阶→高级的进阶指南 |
| 常见问题 | 常见错误及解决方案 |

## 学习路径

| 阶段 | 时间 | 目标 |
|------|------|------|
| 初学 | 0-6 个月 | 读谱、基本节奏、简单曲目 |
| 初级 | 6-18 个月 | 快速读谱、八度音阶 |
| 中级 | 1.5-3 年 | 复杂视奏、练习曲 |
| 高级 | 3+ 年 | 奏鸣曲、协奏曲 |

## 推荐资源

- **教材**: Alfred's, Bastien, Faber
- **练习曲**: Hanon, Czerny
- **曲目**: 巴赫小步舞曲、月光奏鸣曲
- **网站**: Musictheory.net
